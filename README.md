## Overview
Item2Vecを用いた映画のレコメンダサービスを作りました。オフライン学習させたWord2VecをExpress.jsとPytthonを用いてAPI経由で稼働させています。フロントエンドはNext,jsとReact Three Fiber & Dreiを用いています。画面右上のEishinIshida(自分の名前)を押すと私のポートフォリオサイトに飛ぶようになっています。

## Usage
・frontendディレクトリでnpm installを実行後、npm run devでフロントエンドを動かすことができます。

・映画ポスターの取得にBing image search APIを用いており、そのサブスクリプションキーをここでは伏せているためバックエンドの処理はできません。このファイルを保存して実行できるのはフロントエンドのみになります。

・学習に用いたデータセットはファイルサイズが大きすぎるため、整形した一部の学習用データのみgithubにアップしています。

## Features
私は現在Webメタバースの開発を行っており、その一部である3D空間をアバターが移動できるシステムをこのプロジェクトのポートフォリオの部分に搭載しました。初めてこのサイトを見た人を驚かせられるようなギミック作りが工夫ポイントの一つです。
